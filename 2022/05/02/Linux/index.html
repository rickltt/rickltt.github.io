Linux简单教程

Linux常规文件目录

Linux文件系统结构是从Unix文件结构演进过来的。在Linux文件系统中，通用的目录名用于表示一些常见的功能，下面列出了一些较常见的Linux顶层虚拟目录名及其内容。

Linux常用命令

注意：这里列举了常用的Linux命令，不包含全部内容，如果你想更加深入学习，可以去查找相关资料。在Linux中可以使用man 命令来查看相应的命令手册，比如man cp即可查看cp命令的说明手册，上面写的很详细。

注意：linux命令和参数之间要有空格

遍历目录

  • cd destination ： destination表示想切换到的目录名。destination参数可以用两种方式表示：一种是使用绝对文件路径，另一种是使用相对文件路径。如果不了解绝对路径和相对路径，可以参考绝对路径和相对路径详解。
    • cd ~ 进入用户(user)目录
    • cd / 进入根(root)目录

此外，可以使用单点符（.），表示当前目录； 双点符（..），表示当前目录的父目录.可以这样使用：

  • cd .. 返回上一级目录
  • cd ../ 再加上，上一级目录下其他文件夹的名字，就可以切换到那个路径。

文件和目录列表

  • pwd 查看当前路径
  • ls ：列出当前路径下的文件和文件夹
    • ls -F 区分文件和目录
    • ls -a 隐藏文件和普通文件及目录一起显示
    • ls -l 长列表格式输出，文件类型，文件权限，文件的硬链接总数，文件属主的用户名，文件属组的组名，文件的大小（以字节为单位），文件的上次修改时间，文件名或目录名等信息。
      • 过滤输出列表：ls命令能够识别标准通配符，并在过滤器中用它们进行模式匹配。问号（?）代表一个字符,星号（*）代表零个或多个字符。
        • ls -l my_scr?pt 问号可用于过滤器字符串中替代任意位置的单个字符
        • ls -l my* 星号可匹配零个或多个字符
      • 使用中括号：
        • ls -l my_scr[ai]pt 使用了中括号以及在特定位置上可能出现的两种字符：a或i进行模式匹配。
        • ls -l f[a-i]ll 字母范围[a – i]进行模式匹配。
        • ls -l f[!a]ll 使用感叹号（!）将不需要的内容排除在外。
    • ls -R 列出了当前目录下包含的子目录中的文件，-R参数表示递归选项。
    • ls -alh 可以这样组合使用，h和l组合时，会以MB、GB等方式显示文件大小。

处理文件和目录

  • touch FILE 创建一个名为FILE的空文件

  • cp <source> <destination> 复制文件，cp命令需要两个参数<source> 、<destination>分别表示源对象和目标对象。

    • cp -i 询问是否需要覆盖已有文件
    • cp -R 递归地复制整个目录的内容
    • cp file . 单点符（.）就很适合用于cp命令。记住，单点符表示当前工作目录
    • 更多细节可以用man cp查看命令手册。

  • mv 移动文件

    • mv <oldfile> <newfile> 重命名
    • mv <source> <destination> 移动文件位置
    • mv -i 和 cp 命令类似，也可以在 mv 命令中使用 -i参数 。这样在命令试图覆盖已有的文件时，你就会得到提示.

  • rm 删除文件

    • rm -r 递归删除
    • rm -d 删除目录
    • rm -i 删除前有提示
    • rm -f 删除前没提示

  • mkdir 新建目录

  • rmdir 删除目录

查看文件内容

  • file 查看文件类型
  • cat 查看文件内容
    • cat -n 所有行加上行号
    • cat -b 给有文本的行加行号，空行不加行号
  • tail
    • tail -n 显示后面n行，默认是10
  • head
    • head -n 显示前面n行，默认是10

  • ps -A 列出所有的进程
  • ps -ef
  • ps -w 显示加宽可以显示较多的资讯

压缩和解压

  • zip 压缩为zip文件
    • zip -q -r FILE.zip DIR/ 将DIR目录下的文件压缩成FILE.zip ，-q 不打印压缩过程，-r递归处理，将DIR目录下的文件和目录递归压缩
  • unzip 解压zip文件
    • unzip file.zip -d DIR/ 压缩到指定目录,file是你的文件名
  • tar
    • -j 将输出重定向给bzip2命令来压缩内容
    • -x 将输出重定向给gzip命令来压缩内容
    • -c 创建一个新的tar归档文件
    • -t 列出已有tar归档文件的内容
    • -v 在处理文件时显示文件
    • -f 输出结果到文件或设备
    • tar -zcvf file.tar.gz DIR/FILE 压缩DIR目录下的文件,file是你指定的文件名。
    • tar -ztvf file.tar.gz 查询
    • tar -zxvf file.tar.gz -C DIR 解压文件到DIR目录下
    • 如果是.bz2,则把参数z换成j

      如果以上看不懂，遇到tar.gz结尾的压缩文件，直接用tar -zxvf 压缩文件名.tar.gz就可以解压该文件了，前面加上路径的话就可以解压到指定文件夹。

监测磁盘空间

  • df ：有时你需要知道在某个设备上还有多少磁盘空间。df命令可以让你很方便地查看所有已挂载磁盘的使用情况。

    • df -h 按照用户易读的形式显示，通常用M来替代兆字节，用G替代吉字节。

  • du : du命令可以显示某个特定目录（默认情况下是当前目录）的磁盘使用情况。

    • du -h: 按用户易读的格式输出大小，即用K替代千字节，用M替代兆字节，用G替代吉字节。
    • du -c: 显示所有已列出文件总的大小。
    • du -s: 显示每个输出参数的总计。

  • mount

  • unmount

查看端口占用

  • netstat:
    • -t: 显示tcp相关选项
    • -u: 显示udp相关选项
    • -n:-拒绝显示别名，能显示数字的全部转化为数字
    • l: 仅列出在Listen(监听)的服务状态
    • p: 显示建立相关链接的程序名

netstat -tunlp | grep 端口号

文件权限

每个Linux文件具有四种访问权限：可读(r)、可写(w)、可执行(x)和无权限(-)。
利用ls -l命令可以看到某个文件或目录的权限，它以显示数据的第一个字段为准。第一个字段由10个字符组成，如下：

-rwxr-xr-x

  • 第一位表示文件类型，-表示文件，d表示目录
  • 后面有三组rwx，分别代表 文件所有者（Owner）、用户组（Group）、其它用户（Other Users）的权限。

修改文件权限

数字表示法： 是指将r、w和x分别用4、2、1来代表，没有授予权限的则为0，然后把权限相加

文本表示法：4个字母表示不同的用户

  • u：所有者
  • g：组成员
  • o：其他成员
  • a：所有人

权限仍用r、w和x表示

  • =：重新制定权限
  • -：对目前的设置减少权限
  • +：对目前的设置增加权限

Linux组

用户账户在控制单个用户安全性方面很好用，但涉及在共享资源的一组用户时就捉襟见肘了。为了解决这个问题，Linux系统采用了另外一个安全概念——组（group）。

组权限允许多个用户对系统中的对象（比如文件、目录或设备等）共享一组共用的权限。

/etc/group

与用户账户类似，组信息也保存在系统的一个文件中，该文件中每一行的格式如下：

  • groupadd:创建新组
  • groupmod:修改组
